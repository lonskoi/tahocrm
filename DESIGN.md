# Дизайн-система tahoCRM

## Премиальный визуальный дизайн

Проект tahoCRM использует современную, премиальную дизайн-систему, созданную для максимального визуального впечатления и удобства использования.

## Цветовая палитра

### Основные цвета

- **Primary (Синий)**: `#3b82f6` → `#2563eb` (градиент)
- **Secondary (Фиолетовый)**: `#8b5cf6`
- **Accent (Циан)**: `#06b6d4`
- **Success (Зеленый)**: `#10b981`
- **Warning (Оранжевый)**: `#f59e0b`
- **Error (Красный)**: `#ef4444`

### Градиенты

- **Primary Gradient**: `from-blue-600 to-purple-600`
- **Secondary Gradient**: `from-purple-600 via-blue-600 to-cyan-600`
- **Accent Gradient**: `from-blue-500 to-cyan-500`

## Компоненты

### Button

- 5 вариантов: `default`, `outline`, `ghost`, `destructive`, `gradient`
- 4 размера: `sm`, `md`, `lg`, `icon`
- Анимации hover и tap через Framer Motion
- Состояние загрузки с spinner
- Градиентные варианты с тенями

### Input

- Плавные переходы фокуса
- Иконки слева
- Состояния ошибок
- Анимации при появлении
- Тени при фокусе

### Modal

- Backdrop blur эффект
- Плавные анимации появления/исчезновения
- Адаптивные размеры
- Закрытие по клику вне модалки

### Card

- Hover эффекты с поднятием
- Градиентные варианты
- Тени с глубиной
- Скругленные углы (16px)

## Анимации

### Framer Motion

- Плавные переходы страниц
- Анимации появления элементов
- Hover и tap эффекты
- Stagger анимации для списков

### Кастомные анимации

- `fadeIn` - появление с прозрачностью
- `slideIn` - появление сбоку
- `scaleIn` - масштабирование
- `shimmer` - эффект блеска

## Типографика

- **Заголовки**: Bold, крупные размеры (2xl-4xl)
- **Текст**: Medium weight для важного, Regular для обычного
- **Градиентный текст**: Для акцентов и логотипов
- **Иерархия**: Четкое разделение уровней

## Эффекты

### Glass Morphism

- Полупрозрачный фон с blur
- Используется в модалках и хедере

### Тени

- Многоуровневая система теней
- Цветные тени для градиентных элементов
- Hover эффекты с увеличением тени

### Градиенты

- Используются для кнопок, карточек, фонов
- Плавные переходы цветов
- Создают глубину и премиальность

## Страницы

### Login

- Анимированный фон с blob эффектами
- Glass morphism карточка
- Градиентный логотип
- Плавные анимации полей ввода

### Dashboard

- Статистические карточки с градиентами
- Иконки с цветными фонами
- Hover эффекты на всех интерактивных элементах
- Адаптивная сетка

### Tasks

- Цветовая кодировка статусов
- Плавные переходы между состояниями
- Иконки для визуального разделения
- Карточки с border accent

## Адаптивность

- Mobile-first подход
- Адаптивная навигация (мобильное меню)
- Гибкие сетки (grid)
- Оптимизация для всех размеров экранов

## Интерактивность

- Все кнопки с hover и tap эффектами
- Плавные переходы состояний
- Визуальная обратная связь на все действия
- Анимации загрузки

## Доступность

- Focus states для клавиатурной навигации
- Контрастные цвета для текста
- Семантический HTML
- ARIA атрибуты где необходимо

## Производительность

- Оптимизированные анимации (GPU ускорение)
- Lazy loading для тяжелых компонентов
- Минимальные перерисовки
- Эффективное использование Framer Motion

## Уникальные фичи

1. **Универсальная кнопка "+"** - плавающая с градиентом и анимацией
2. **Smart Search** - с выпадающими результатами
3. **Статистические карточки** - с градиентными иконками
4. **Анимированный фон** - на странице логина
5. **Glass morphism** - современный эффект прозрачности

## Технологии дизайна

- **Tailwind CSS 4** - утилитарный CSS
- **Framer Motion** - анимации
- **Lucide React** - иконки
- **Custom CSS** - кастомные эффекты

## Результат

Дизайн создает впечатление премиального, современного продукта стоимостью в миллионы долларов, с вниманием к деталям, плавными анимациями и профессиональным визуальным оформлением.
